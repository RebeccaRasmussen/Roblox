# Game Projects

This folder will contain your individual game projects!

## Organizing Your Games

Each game should have its own folder with:
- **README.md**: Description of the game
- **Scripts**: Lua scripts for the game
- **Notes**: Ideas, plans, and progress notes

## Example Structure

```
games/
├── my-first-obby/
│   ├── README.md
│   ├── main-script.lua
│   └── notes.md
├── coin-collector/
│   ├── README.md
│   └── scripts/
└── adventure-quest/
    ├── README.md
    ├── scripts/
    └── design-docs/
```

## Game Ideas for Beginners

### Easy Projects
- **Obby (Obstacle Course)**: Jump across platforms to reach the end
- **Clicker Game**: Click a button to earn points
- **Color Changer**: Parts that change color when touched
- **Teleporter**: Pads that move you to different locations

### Intermediate Projects
- **Tycoon Game**: Build and upgrade a business
- **Racing Game**: Cars and a race track with checkpoints
- **Sword Fighting**: Combat system with weapons
- **Pet Simulator**: Collect and level up pets

### Advanced Projects
- **RPG Adventure**: Quests, NPCs, and inventory system
- **Tower Defense**: Defend against waves of enemies
- **Simulator Game**: Complex progression and upgrades
- **Battle Royale**: Multiplayer combat game

## Tips

- Start each game in a new Roblox Studio place
- Document your ideas before coding
- Break big games into small features
- Test each feature before moving on
- Don't be afraid to start over if needed!

Ready to create your first game? Make a new folder and get started! 🚀
